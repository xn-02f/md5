## Install

Via npm :

```bash
npm i @xn-02f/md5
```

## Usage

```javascript
import md5 from '@xn-02f/md5'

md5('xn-02f');  // => '54d30fa674d13e3598970bc9c5e2388e'
```

Parameter | return
-- | --
`string`<br>The value to be converted. | `string`<br>MD5 hex hash string.
